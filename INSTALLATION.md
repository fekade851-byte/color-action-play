# Color Move Kids - Installation Guide

## Overview
Color Move Kids is a Progressive Web App (PWA) that works on phones, tablets, and Android Smart TVs.

## Installation Methods

### For Mobile Phones (iOS & Android)

#### Option 1: Add to Home Screen (Recommended)
1. Open the website in your mobile browser (Chrome, Safari, Firefox, etc.)
2. Tap the browser menu (three dots or share icon)
3. Select "Add to Home Screen" or "Install App"
4. Tap "Add" or "Install"
5. The app icon will appear on your home screen

#### Option 2: Direct Browser Access
Simply open the website URL in your mobile browser and bookmark it for easy access.

### For Android Smart TV

#### Option 1: TV Browser
1. Open the built-in web browser on your Smart TV (Chrome, Firefox, or Samsung Internet)
2. Navigate to the website URL
3. Bookmark the page for easy access
4. Use your TV remote to navigate and play

#### Option 2: Chrome for Android TV
1. Install Chrome browser from Google Play Store on your TV
2. Open Chrome and navigate to the website
3. The app will work in fullscreen mode
4. Use your remote control to interact

#### Option 3: Cast from Phone
1. Install the app on your phone (see above)
2. Make sure your phone and TV are on the same WiFi network
3. Use Chrome's cast feature to display on TV
4. Control the app from your phone while viewing on TV

## Features

- Animated placeholders always visible
- Adjustable game duration (1-30 minutes)
- Adjustable video duration (3-15 seconds)
- Adjustable number of rounds (1-10)
- Auto-play mode
- Full audio support
- Offline support via service worker
- Optimized for Smart TV displays (1080p/4K)
- Touch-friendly controls for mobile
- Remote-control friendly for TV

## Requirements

### Minimum Requirements
- Modern web browser (Chrome 80+, Safari 13+, Firefox 75+)
- Internet connection for first load
- Screen resolution: 720p minimum (1080p recommended)

### Recommended for Smart TV
- Android TV 8.0 or higher
- 1080p or 4K display
- 2GB RAM minimum
- Bluetooth remote or air mouse for navigation

## Troubleshooting

### Audio Not Playing
- Make sure device volume is up
- Check that browser allows autoplay with sound
- For iOS: tap the screen once to enable audio
- For Smart TV: check TV audio settings

### Videos Not Loading
- The app will show animated placeholders if videos fail to load
- Check internet connection
- Clear browser cache and reload
- Ensure video files are in the correct format (MP4/H.264)

### App Not Installing on Mobile
- Clear browser cache
- Update browser to latest version
- Try a different browser
- Check device storage space

### Performance Issues on Smart TV
- Close other apps running in background
- Clear TV browser cache
- Reduce game duration in settings
- Ensure TV has good WiFi signal

## Controls

### Mobile/Tablet
- Tap buttons to navigate
- Use settings to customize experience
- Swipe gestures supported on some devices

### Smart TV Remote
- Use D-pad to navigate menus
- Press OK/Enter to select
- Press Back to return to previous screen
- Use number keys for quick settings access (if available)
